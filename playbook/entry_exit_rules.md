# エントリー / イグジット ルール

## エントリー条件
### 必須条件（全て満たすこと）
1. ファンダメンタルズで割安であること（PER/PBR同業比較、売上・利益成長率 vs 時価総額）
2. ビジネスモデルを理解していること（セグメント別の売上・利益貢献を把握）
3. テクニカルでエントリータイミングが来ていること（移動平均線・ボリンジャーバンド）
4. 需給環境が悪くないこと（信用買い残が異常に多くないか確認）
5. 感情的な判断ではないこと（FOMO・リベンジトレードでないか自問）

### 加点条件（満たすほど確信度UP）
1. 信用売り残が多い or 増加傾向（需給良好）
2. 機関投資家の空売り残が減少傾向
3. 自己資本比率が健全、CFが安定
4. カタリスト（決算、新規事業、業界追い風等）が見込める
5. 業界全体に資金が集中している

## イグジット条件
### 利確ルール
- ボリンジャーバンド+3σタッチ（特に5分足・日足）→ **利確を強く検討**
  - PMの弱点：「上昇を逃したくない」心理で利確できない → ここはエージェントが最優先で警告
- 目標株価に到達
- ファンダメンタルズの前提が崩れた場合（業績下方修正等）

### 損切りルール
- 事前に設定した損切りラインに到達 → **厳守（例外なし）**
- ファンダメンタルズの前提が崩れた場合 → 損切りラインに関わらず撤退検討
