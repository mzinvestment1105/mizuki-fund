# 個別銘柄選定の基準

## スクリーニング条件
### 定量条件
| 指標 | 条件 | 備考 |
|------|------|------|
| PER | 同業他社比較で割安 | 絶対値よりも相対比較を重視 |
| PBR | 同業他社比較で割安 | 1倍割れは特に注目 |
| 売上成長率 | プラス成長が望ましい | 時価総額との比較が重要 |
| 営業利益成長率 | プラス成長が望ましい | 売上成長率との乖離に注目 |
| 自己資本比率 | 業界平均以上 | 低すぎる場合は財務リスク |
| CF | 営業CF > 投資CF | FCFがプラスであること |
| 信用買い残 | 以下の3軸で評価（小型株は信用倍率単体に意味なし） | 詳細は下記「信用残の見方」参照 |
| 信用売り残 | 多いほど良い（売りできる銘柄のみ有効）| 踏み上げの可能性 |

### 定性条件
1. ビジネスモデルが理解できること（何で稼いでいるか説明できるレベル）
2. セグメント別の売上・利益貢献度が把握できること（情報が少なければ推測でも可）
3. 過去にワラント等の希薄化を伴う増資がないこと（あれば要注意）

### 信用残の見方（小型株対応）

小型株は信用売りができない銘柄が多いため、**信用倍率単体はほぼ無意味**。以下の3軸で評価する：

| 軸 | 計算式 | 判断基準 |
|----|--------|---------|
| ① 浮動株に対する比率 | 信用買残株数 ÷ 発行済株式総数 | 10%超は重い。20%超は要警戒 |
| ② 解消日数 | 信用買残株数 ÷ 一日平均出来高 | 20日超は重い（需給解消に時間がかかる） |
| ③ トレンド | 直近2ヶ月の信用買残の増減方向 | **減少トレンドは良い兆候**。増加中は需給悪化継続 |

## 銘柄分析で必ず確認すること
- [ ] 財務状況（自己資本比率、CF、有利子負債）
- [ ] バリュエーション（PER/PBR 同業他社比較）
- [ ] 売上・営利・最終益の成長率と時価総額の比較
- [ ] セグメント別の売上・利益構成
- [ ] 大株主の構成と変動状況
- [ ] 過去の増資傾向（特にワラント）
- [ ] カタリスト（決算、新規事業、業界動向）
- [ ] テクニカル（移動平均線、ボリンジャーバンド）
- [ ] 需給環境（信用買残・売残・機関空売り残）

## 買わない銘柄の特徴
1. ビジネスモデルが理解できない
2. 過去に繰り返しワラント等の希薄化増資を行っている
3. 信用買い残が異常に多く需給が悪い
4. FOMO（急騰に乗り遅れたくない）だけが買いたい理由
