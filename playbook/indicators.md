# 重視する指標

## テクニカル指標
| 指標 | 用途 | 見方 |
|------|------|------|
| 移動平均線（日足・週足・月足） | トレンド判断 | 短期線が長期線を上抜け＝上昇トレンド、下抜け＝下落トレンド |
| ボリンジャーバンド | エントリー/イグジットタイミング | +3σタッチ＝過熱（利確検討）、-3σタッチ＝売られすぎ（買い検討） |
| ボリンジャーバンド（5分足） | 短期的な過熱判断 | +3σで飛んだ時は売った方が期待値が高い（**重点サポート対象**） |
| 出来高 | トレンドの信頼度 | 出来高を伴う値動きはトレンドの信頼度が高い |

### ボリンジャーバンド3σに関する特記事項
PMは「上昇を逃したくない」という気持ちから、+3σで飛んだ時に利確行動が取れないことが多い。ここは **全エージェントが最優先で警告すべきポイント** である。

## ファンダメンタルズ指標
| 指標 | 用途 | 見方 |
|------|------|------|
| 売上高・成長率 | 事業成長の把握 | 前年比・四半期推移で成長トレンドを確認 |
| 営業利益・成長率 | 収益力の把握 | 本業の稼ぐ力。売上成長率との乖離に注目 |
| 最終益・成長率 | 純利益の把握 | 特別損益の影響を除いた実力値を見る |
| PER | バリュエーション | 同業他社比較で割安/割高を判断 |
| PBR | バリュエーション | 同業他社比較。1倍割れは注目 |
| 時価総額 vs 利益 | 総合的な割安度 | 売上・営利・最終益の成長率を時価総額と比較 |
| 自己資本比率 | 財務健全性 | 低すぎる場合はリスク。業界平均との比較 |
| CF（キャッシュフロー） | ビジネスの健全性 | 営業CF＞投資CFが基本。FCFの推移 |
| 大株主構成・変動 | ガバナンス・需給 | 大株主の異動、特に機関投資家の動向 |
| 過去の増資傾向 | 希薄化リスク | **特にワラント等がある場合は要注意** |
| セグメント別売上・利益 | 事業構造の理解 | 各事業の貢献度を把握（情報が少なければ推測でも可） |

## マクロ指標（最重要）
| 指標 | 用途 | 見方 |
|------|------|------|
| 日経平均 | 日本株全体のトレンド | 個別銘柄の地合い判断の基本 |
| ダウ / S&P500 | 米国株・グローバル地合い | 米国の動きが翌日の日本株に影響 |
| ドル円為替 | 輸出入企業への影響 | 円安＝輸出有利、円高＝内需有利 |
| ビットコイン | リスクセンチメント | リスクオン/オフの先行指標 |
| 金（Gold） | 安全資産の動向 | 金上昇＝リスクオフ傾向 |
| 金利（米国債・日本国債） | 金融環境 | 金利上昇＝グロース株に逆風、バリュー株に追い風 |

### マクロで見落としてはならない発言・イベント
- **日本総理大臣発言**
- **米国大統領発言**
- **FRB発言（FOMC含む）**
- **日銀発言（金融政策決定会合含む）**
- 経済指標カレンダー（雇用統計、CPI、GDP等）
- その他コモディティ関連ニュース

## センチメント指標
| 指標 | 用途 | 見方 |
|------|------|------|
| 信用買い残（個人） | 需給判断 | 多い/増加＝需給悪い（個人が楽観している） |
| 信用売り残（個人） | 需給判断 | 多い/増加＝需給良い（個人が悲観している） |
| 機関投資家の空売り残 | 参考指標 | 残高少/減少＝ポジティブ。多い/増加＝暴落リスク注意（ヘッジ目的も多いため参考程度） |
| VIX（恐怖指数） | 市場全体の恐怖度 | 高い＝恐怖（逆張り買いの好機の可能性）、低い＝楽観（警戒） |
| 投資部門別売買状況 | 資金フローの把握 | 海外投資家・個人・信託銀行等の売買動向 |
| X（SNS）・掲示板 | 定性的センチメント | 個人投資家が楽観＝売り検討、絶望＝買い検討 |

## 指標の優先順位
テクニカルとファンダメンタルズが矛盾した場合：
1. **ファンダメンタルズが良くテクニカルが悪い** → タイミングを待つ（焦らない）
2. **テクニカルが良くファンダメンタルズが悪い** → 見送り（テクニカルだけでは入らない）
3. **需給が極端に悪い** → ファンダ・テクニカルが良くても慎重に（信用買い残が異常に多い等）
