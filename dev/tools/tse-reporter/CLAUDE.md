# TSE Reporter — Claude Code 指示書

## モデル使用制約

- API呼び出しには必ず `claude-haiku-4-5` を使用すること
- `claude-sonnet-4-6` や `claude-opus-4-6` は**ユーザーから明示的に許可を得た場合のみ**使用可能
- モデルを変更する場合は必ず事前にユーザーに確認すること

## コスト管理

- 実装変更でAPI呼び出し回数やトークン数が増える場合は、事前にコスト影響を説明すること
- リトライロジックを追加する場合は、最悪ケースのコストを計算してユーザーに提示すること
- `web_search` ツールは大量のトークンを消費するため、呼び出し回数を最小限にすること

## 設定ファイル

- モデル名は `src/config.py` の `ANTHROPIC_MODEL` で一元管理する
- ハードコードで別のモデルを指定することを禁止する

## 作業前の確認事項

- コストに影響する変更（モデル変更、API呼び出し増加）は必ず事前承認を得ること
- 本番実行前に小規模テスト（`--max-stocks 1` など）でコストを確認すること
